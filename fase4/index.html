Propósito del sitio

El propósito de este sitio web es proporcionar información acerca de Etiquetas Básicas de HTML, atributos y valores, ya que esto facilitará el diseño de sitios web. Las etiquetas son un código que permite que el diseñador proporcione al navegador la forma en que debe interpretar la información suministrada, ya sea un enlace, un párrafo, un título, etc. Dentro de las etiquetas se encuentran los atributos los cuales permiten controlar la interpretación dada de estas y los valores los cuales permiten identificar la propiedad asignada al atributo.

Etiquetas Básicas de HTML, atributos y valores

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es lenguaje con el cual se define el contenido y el diseño de las paginas web, mediante el uso de etiquetas que permite definir el texto o los diferentes elementos que componen el sitio.

Las etiquetas son un conjunto de caracteres que conforman el formato del sitio web, por lo cual es importante conocer las diferentes etiquetas y su funcionalidad a la hora de diseñar sitios web.
